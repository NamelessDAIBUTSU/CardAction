【**プロジェクト概要**】

UE5を学習するために始めた個人開発用のプロジェクトです。
ジャンルは「グリッドバトル × 3Dカードアクション」で、ロックマンエグゼ × Slay the Spireのようなイメージです。

【**開発環境**】
- Unread Engine 5.4.4
- C++
- Visual Studio 2022
- Git / SourceTree

【**フォルダ構成**】
```
Source/
|———— CardAction/
     |————— Private/ # .cppファイル。Public階層と同構成のため省略
     |...
     |————— Public/  # .hファイル
           |
           |————— AI/ # 汎用的なAI(BehaviorTree)関連のファイル
           |     |————— Controller/ # AIContollerファイル。複数ファイルはないが見た目の統一のため一応フォルダ化
           |     |————— Decorator/  # 汎用的に使えるBTDecoratorファイル
           |     |————— Service/    # 汎用的に使えるBTServiceファイル
           |     |————— Task/       # 汎用的に使えるBTTaskファイル
           |
           |————— Anim/   # 汎用的に使えるアニメーション通知ファイル
           |————— Camera/ # グリッド見下ろし用のカメラファイル
           |————— Card/   # カードやデッキなどカードに関連するファイル
           |      |————— Effect/ # カード効果ファイル
           |
           |————— Character/  # プレイヤー関連ファイル
           |————— Controller/ # レベル毎のプレイヤーコントローラーファイル
           |————— Enemy/      # 敵関連ファイル
           |      |————— CactusEnemy/  # カクタス(敵)関連ファイル
           |      |————— ElectricEye/  # エレクトリックアイ(敵)関連ファイル
           |      |————— PumpkinEnemy/ # パンプ(敵)関連ファイル
           |
           |————— Grid/ # グリッド関連ファイル
           |      |————— Cell/ # グリッドセル関連ファイル
           |
           |————— Interface/ # インターフェースファイル
           |————— Map/       # マップ関連ファイル
           |      |————— Stage/ # ステージ関連ファイル
           |
           |————— MyComponents/ # 自作コンポーネントファイル
           |————— Projectile/   # 弾関連ファイル（オミット予定）
           |————— System/       # ゲームモードやフェードなどシステムに関連するファイル
           |      |————— Phase/   # バトルフェーズ関連ファイル
           |
           |————— UI/ # UI関連ファイル。HUDでもOutGameでも使うウィジェットは直下に配置
           |      |————— HUD/ # HUD関連ファイル
           |      |————— OutGame/ # アウトゲームUI関連ファイル(追加予定)
           |
           |————— Util/ # 汎用的に使うUtil関連ファイル
           |————— Weapon/ # プレイヤーの武器（攻撃）関連ファイル
```
